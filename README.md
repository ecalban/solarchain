# SolarChain

SolarChain, mahalle ölçeğinde güneş enerjisi üreticileri ile tüketicileri buluşturan bir enerji pazarı ve simülasyon uygulamasıdır.

## Özellikler

- Canlı enerji ilanlarını görüntüleme ve enerji satın alma
- Mahalle enerji üretim/tüketim simülasyonu
- Simülasyondaki fazla enerjiden ilan oluşturma
- Üretici, alıcı ve yönetici rolleri
- İlan onaylama ve reddetme işlemleri
- Cüzdan, işlem geçmişi ve temiz enerji etkisi
- Yenilemesiz sayfa geçişleri ve responsive arayüz

## Çalıştırma

Node.js kurulu bir bilgisayarda:

```bash
git clone https://github.com/ecalban/solarchain.git
cd solarchain
npm start
```

Ardından `http://127.0.0.1:3000` adresini açın.
